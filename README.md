# ha-mqtt
Generate MQTT entities in Home Assistant

[README.md in Chinese](README.zh.md)

Reference document: https://www.home-assistant.io/docs/mqtt/discovery/

## Supported languages
- ✔️ English (en-US)
- ✔️ Chinese (zh)
## Supported components
- ✔️ alarm_control_panel
- ✔️ binary_sensor
- ✔️ button
- ✔️ camera
- ✔️ climate
- ✔️ cover
- ✔️ device_tracker
- ✔️ device_automation
- ✔️ fan
- ✔️ humidifier
- ✔️ light
- ✔️ lock
- ✔️ number
- ✔️ scene
- ✔️ select
- ✔️ sensor
- ✔️ switch
- ✔️ tag
- ✔️ vacuum

Auto-discovery
```yaml
topic: homeassistant/status
payload: online
```
